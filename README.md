In interactive mode, the last printed expression is assigned 

to the variable _. 

This means that when you are using Python as a desk calculator, 

it is somewhat easier to continue calculations, for example:

tax = 12.5 / 100

price = 100.50

price * tax

price + _

round(_, 2)
